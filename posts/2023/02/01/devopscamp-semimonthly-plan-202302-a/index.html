第一期回顾

1. 参与者

有两位朋友参与作业提交

  1. calacaly
  2. zhangdongdong7

2. 优化点

  1. 将作业项目独立之后， 确实方便编译了， 但是同时也与主站隔离了， 无法方便的展示所有数据。 所有操作都需要跳转。 这个并不是我想要实现的分享社区。

关于这个问题， 我继续还要继续研究。

  1. 现在标题同名， 无法区分。

我已经有想法了， 会通过工具在合并的时候， 强制加上各位的 Github 名称。 至于各位标题的实际名称， 各位随意。

注： Github 名称， 就是各位作业路径名称。

3. 关于分享展示

鼓励各位在 Markdown 中添加自己的 博客地址， 公众号地址 等等， 大家认识你。

之前我考虑在标题上帮大家加上， 后来仔细想想， 并不妥。 如果固定在这里， 那么字段数量必然固定， 并不利于分享。

新的建议， 大家在分享的时候， 直接写在 markdown 的 头部 或者 尾部， 二选一。

当然也可以不添加， 这个全凭自愿。

4. 关于作业提交

关于作业怎么分享 这里有 B 站视频 1.5倍数观看， 效果应该最好。
